Interview Ready AI

Interview Ready AI is a service that helps candidates prepare for job interviews and optimize their resumes using AI.
The system analyzes a candidate's resume, self-description, and a job description to generate:

1. Interview preparation reports
2. Technical and behavioral interview questions
3. Skill gap analysis
4. A preparation roadmap


Tech Stack

Backend 
  Node.js
  Express.js
  MongoDB + Mongoose
  Groq AI API

Frontend
  Reactjs
  Axios
  Multer

Installation

Clone the repository:

git clone https://github.com/yourusername/interview-ready-ai.git
cd interview-ready-ai/backend

Install dependencies

Environment Variables

Create a .env file in the root directory.

PORT=3000
MONGO_URI=your_mongodb_connection
GROQ_API_KEY=your_groq_api_key
JWT_SECRET_KEY=your_jwt_secret_key

Running the Server

Start the development server:

npm run dev

or

node server.js

Server will run on:

http://localhost:3000
